# BioMarket

Aplicacion web cliente-servidor desarrollada con Spring Boot, Spring Data JPA, MySQL, Thymeleaf y Bootstrap.

## Documentacion API

La documentacion Swagger/OpenAPI esta disponible cuando la aplicacion esta en ejecucion:

- Swagger UI: http://localhost:8080/swagger-ui.html
- OpenAPI JSON: http://localhost:8080/v3/api-docs

Tambien se incluye una copia exportada en:

- `docs/api/openapi.json`

## Manual de usuario

- Video explicativo: https://youtu.be/hAC5oDGMVIo
- Manual de usuario: `docs/user-manual.md`

## Modulos principales

- CRUD de productos e inventario.
- Carrito de compras.
- Registro de pedidos con cliente, detalle, total y validacion de stock.
- Panel administrativo de productos y pedidos.
